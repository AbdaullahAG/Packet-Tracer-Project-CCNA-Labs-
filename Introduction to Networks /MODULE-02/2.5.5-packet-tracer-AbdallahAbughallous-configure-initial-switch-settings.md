Why is the login command required?

Step 7: Verify that the enable secret password is added

What is displayed for the enable secret password?
شيء مشابه لـ: $1$mERr$ILwq/b7kc.7X/ejA4Aosn0 (شكل مشفر).
Why is the enable secret password displayed differently from what we configured?
لأن الـ enable secret يستخدم دائماً تشفيراً قوياً (MD5 hashing) بشكل تلقائي، بينما الـ enable password يكون نصاً عادياً (plain text) إلا إذا تم تشفيره يدوياً.

Step 8: Encrypt the enable and console passwords

If you configure any more passwords on the switch, will they be displayed in the configuration file as plain text or in encrypted form? Explain.
سيتم عرضها في شكل مشفر (encrypted form).
الأمر service password-encryption يشفر جميع كلمات المرور الحالية والمستقبلية (بما فيها console و enable password و vty إلخ) باستخدام خوارزمية تشفير بسيطة (نوع 7). لن تظهر أي كلمة مرور جديدة كنص عادي بعد تنفيذ هذا الأمر.

Part 3: Configure a MOTD Banner
Step 1: Configure a message of the day (MOTD) banner

When will this banner be displayed?
يظهر الـ banner عندما يقوم أي شخص بالدخول إلى الـ switch عبر الـ console port أو عبر الـ VTY (Telnet/SSH) قبل ظهور شاشة User Access Verification.
Why should every switch have a MOTD banner?
لتحذير المستخدمين غير المصرح لهم بأن الدخول ممنوع (قانونياً)، ولإرسال رسائل مهمة إلى الفنيين أو المسؤولين عن الشبكة. كما أنه يساعد في توثيق الملكية والتحذير من الوصول غير المصرح به.

What is the shortest, abbreviated version of the copy running-config startup-config command?

Which command will display the contents of NVRAM?

Are all the changes that were entered recorded in the file?
نعم، يجب أن تكون جميع التغييرات (hostname، passwords، banner، service password-encryption...) مسجلة في ملف الـ startup-config تماماً كما في الـ running-config.

Part 5: Configure S2
قم بتكوين الـ Switch S2 بنفس الطريقة التي اتبعتها مع S1، مع التالي:

Device name: hostname S2
Console password:textline console 0
password letmein
login
Privileged mode passwords:textenable password c1$c0
enable secret itsasecret
MOTD Banner (رسالة مناسبة):textbanner motd "This is a secure system. Authorized Access Only!"(يمكنك تغيير النص إذا أردت، لكن يجب أن يكون تحذيرياً).
Encrypt all plain text passwords:textservice password-encryption
تحقق من التكوين:
show running-config
احفظ التكوين:
copy running-config startup-config (أو cop r s)

نصيحة: بعد الانتهاء من S2، تأكد من أنك تستطيع الخروج (exit) والدخول مرة أخرى باستخدام كلمات المرور المعينة (console: letmein، enable secret: itsasecret).
