# Packet Tracer - Connect the Physical Layer

## 📖 Lab Overview
This activity explores the hardware characteristics of Cisco internetworking devices. You will identify management ports, understand expansion slots, select and install appropriate interface modules, and establish physical connectivity between various LAN and WAN devices.

---

## 🛠 Part 1 & 2: Physical Analysis (Questions)

### Device Characteristics
* **East Router Management Ports:** Console and Aux ports.
* **East Router Interfaces:** 2 Gigabit Ethernet (LAN) and 2 Serial (WAN) interfaces.
* **Physical Interfaces (CLI):** 4 physical interfaces are listed (excluding VLAN1).
* **Default Bandwidth:** * GigabitEthernet: 1,000,000 Kbit/sec ($1 Gbps$)
    * Serial: 1,544 Kbit/sec ($1.544 Mbps$)
* **Expansion Slots:** * **East Router:** 1 available slot.
    * **Switch2:** 5 available slots.

### Module Selection
1.  **Module for PC Connectivity:** `HWIC-4ESW` (adds 4 Ethernet switching ports).
2.  **Host Capacity:** 4 additional hosts.
3.  **Fiber Module for Switch2:** `PT-SW-NM-1FGE` (Gigabit Optical connection).

> **Note:** Always power off the device in the **Physical** tab before dragging and dropping new modules into the expansion slots.

## ✅ Part 4: Connectivity Checklist

### 1. Interface Verification
- [ ] Run `show ip interface brief` on **East**.
- [ ] Verify `G0/0`, `G0/1`, `S0/0/0`, and `F0/1/0` through `F0/1/2` are all **Up/Up**.

### 2. Wireless & Cellular Setup
- [ ] **Laptop:** Enable `Wireless0` port status $\rightarrow$ Browse to `www.cisco.pka`.
- [ ] **TabletPC (WiFi):** Enable `Wireless0` port status $\rightarrow$ Browse to `www.cisco.pka`.
- [ ] **TabletPC (4G):** Disable `Wireless0`, Enable `3G/4G Cell1` $\rightarrow$ Verify web access.

### 3. Final Test
- [ ] Ensure all PCs can ping each other and access the web server.
