# 🌐 Network Engineering & Calculation Tools

A curated list of essential online tools for network administrators and engineers for subnetting, conversions, and performance testing.

---

## 1. Subnetting & IP Planning
These tools help calculate IP ranges, subnet masks, and CIDR notation for IPv4 and IPv6.

* **[IPCalculator.net](https://www.ipcalculator.net/)** – A fast, modern calculator for IPv4/IPv6 subnets and wildcards.
* **[Subnet-Calculator.com](http://www.subnet-calculator.com/)** – A classic tool for Classful and Classless (CIDR) subnetting.
* **[CIDR.xyz](https://cidr.xyz/)** – An interactive visualizer for CIDR ranges (great for understanding how IP blocks split).

## 2. Number Systems & Converters
Essential for converting between Hexadecimal, Binary, and Decimal formats.

* **[RapidTables - Hex to Binary](https://www.rapidtables.com/convert/number/hex-to-binary.html)** – Includes step-by-step mathematical explanations for every conversion.
* **[BinaryHexConverters](https://www.binaryhexconverters.com/)** – Simple, dedicated tools for flipping between any base (2, 8, 10, 16).

## 3. Bandwidth & Performance
Calculate data transfer speeds and network overhead.

* **[Omni Calculator - Bandwidth](https://www.omnicalculator.com/other/bandwidth)** – Calculates how long it takes to move specific file sizes over different connection speeds.
* **[Calculator.net - Bandwidth](https://www.calculator.net/bandwidth-calculator.html)** – Converts units (e.g., Mbps to GB/s) and estimates download/upload times.

## 4. Online Network Diagnostics
Cloud-based tools to test connectivity and DNS from outside your local network.

* **[Ping.eu](https://ping.eu/)** – A one-stop shop for Online Ping, Traceroute, DNS lookup, and Port checking.
* **[MXToolbox](https://mxtoolbox.com/)** – The gold standard for checking DNS records, SMTP, and blacklists.
* **[DNSChecker.org](https://dnschecker.org/)** – Checks DNS propagation across global servers to see if your records have updated worldwide.

## 5. Mobile Utilities (On-the-go)
Best apps for field engineers.

* **PingTools Network Utilities** – A comprehensive suite of network tools for your phone.
* **NetCalc** – A lightweight subnet calculator for quick reference during site surveys.

---
> **Tip:** If you are preparing for certifications like **CCNA** or **CompTIA Network+**, use these tools to verify your manual calculations rather than relying on them exclusively!
