Packet Tracer - Configure Initial Router Settings 🚀
📖 OverviewThis lab covers the essential steps for initializing a Cisco router. Key tasks include establishing a console connection, securing the management plane with encrypted passwords, configuring system banners, and managing configuration files in NVRAM and Flash.🎯 ObjectivesPart 1: Verify the Default Router Configuration.Part 2: Configure and Verify the Initial Router Configuration.Part 3: Save the Running Configuration File.🛠️ Configuration Steps1. Initial AccessConnection: Used a Console cable from PCA (RS 232) to R1 (Console).Terminal Access: PCA > Desktop > Terminal > OK.2. Router Hardening (CLI Commands)BashRouter> enable
Router# configure terminal

# Set Hostname
Router(config)# hostname R1

# Secure Privileged EXEC Mode
R1(config)# enable password cisco
R1(config)# enable secret itsasecret

# Secure Console Access
R1(config)# line console 0
R1(config-line)# password letmein
R1(config-line)# login
R1(config-line)# exit

# Configure Legal Banner (MOTD)
R1(config)# banner motd "Unauthorized access is strictly prohibited."

# Encrypt all clear-text passwords
R1(config)# service password-encryption
3. Verification & SavingBash# Verify configurations
R1# show running-config

# Save to NVRAM
R1# copy running-config startup-config

# Optional: Backup to Flash
R1# copy startup-config flash
