## Part 1: Design an IP Addressing Scheme

### Step 1: Subnet the 192.168.100.0/24 network.

**a. How many subnets are needed?** 5 subnets (4 LANs + 1 WAN).

**b. How many bits must be borrowed?** 3 bits ($2^3 = 8 \ge 5$).

**c. How many subnets does this create?** 8 subnets.

**d. How many usable hosts per subnet?** 30 usable hosts ($2^5 - 2 = 30$).

**e. Binary value for the first five subnets:**

## Part 2: Configuration Scripts (CLI)

### Step 1: Configure R1 LAN interfaces
```bash
R1> enable
R1# configure terminal
R1(config)# interface g0/0
R1(config-if)# ip address 192.168.100.1 255.255.255.224
R1(config-if)# no shutdown
R1(config-if)# exit
R1(config)# interface g0/1
R1(config-if)# ip address 192.168.100.33 255.255.255.224
R1(config-if)# no shutdown
```
### Step 2: Configure S3 (Switch Addressing & Gateway)
```bash
S3> enable
S3# configure terminal
S3(config)# interface vlan 1
S3(config-if)# ip address 192.168.100.66 255.255.255.224
S3(config-if)# no shutdown
S3(config-if)# exit
S3(config)# ip default-gateway 192.168.100.65
```
