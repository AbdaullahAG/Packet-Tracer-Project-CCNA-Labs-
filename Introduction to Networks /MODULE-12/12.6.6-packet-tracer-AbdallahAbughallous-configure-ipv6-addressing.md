## 2. Part 1: Configure IPv6 Addressing on the Router

The following configurations were applied to **R1** to enable IPv6 and set up the interfaces:

```bash
R1> enable
R1# configure terminal
R1(config)# ipv6 unicast-routing

# GigabitEthernet0/0 Configuration
R1(config)# interface g0/0
R1(config-if)# ipv6 address 2001:db8:1:1::1/64
R1(config-if)# ipv6 address fe80::1 link-local
R1(config-if)# no shutdown

# GigabitEthernet0/1 Configuration
R1(config)# interface g0/1
R1(config-if)# ipv6 address 2001:db8:1:2::1/64
R1(config-if)# ipv6 address fe80::1 link-local
R1(config-if)# no shutdown

# Serial0/0/0 Configuration
R1(config)# interface s0/0/0
R1(config-if)# ipv6 address 2001:db8:1:a001::2/64
R1(config-if)# ipv6 address fe80::1 link-local
R1(config-if)# no shutdown
R1(config-if)# end
R1# copy running-config startup-config
```
