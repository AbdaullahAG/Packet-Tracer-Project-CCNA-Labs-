CCNA Preparation Labs - Cisco Packet Tracer

This repository contains hands-on labs I completed using Cisco Packet Tracer while preparing for the CCNA certification.

The labs cover key topics from the CCNA syllabus, including:
- Basic device configuration
- VLANs and trunking
- Routing protocols (Static, OSPF, EIGRP)
- ACLs and NAT
- DHCP, SNMP, and more

## You Can Cheack The Labs In Folders Each Lab has .pkt file + screenshot

## How to Use
- Download and open the .pkt files in Cisco Packet Tracer (version 8.x recommended).
- Each lab includes successful configuration and testing (ping, traceroute, etc.).
